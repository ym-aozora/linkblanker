Link Blanker v1.0.3
===============

Overview
----------
リンクを新しいWindow(Tab)で開くGoogle Chrome Extensionです。

Features
----------
* Webページのリンクをクリックした際に新しいWindow(Tab)で開きます。
* ドメイン・ディレクトリ・ページ単位で無効にすることができます。
* 複数クリックすることでタブをまとめて閉じることができます。
