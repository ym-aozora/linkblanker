Link Blanker
===============

Overview
----------
リンクをクリックした際に別タブで開くGoogle Chrome Extensionです。

Features
----------
* Webページのリンクをクリックした際に別タブで開きます。
* 別タブで開く際にバックグラウンドで開く事もできます(要設定)。
* ドメイン・ディレクトリ・ページ単位で無効にすることができます。
* 表示中のWebページ内をトリプルクリックすることで左右どちらかのタブをまとめて閉じることができます。
